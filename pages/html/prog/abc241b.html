長さがそれぞれ\(A_1, A_2, A_3, \cdots ,A_N\) のN本の麺があり、
1日目には長さ\(B_1\), 2日目には長さ\(B_2\), 3日目には長さ\(B_3\),\(\cdots\), M日目には長さ\(B_M\)の麺を食べたい。
ここで\(i\)日目には長さ\(B_i\)ちょうどの麺を食べる必要があるということに注意です。
長さが\(B_i\)を超えてはいけません。
\(M\)日の間すべての日で計画通りに長さ\(B_i\)の麺を食べる必要があります。ここで注意すべき条件は全ての日において計画通りの長さの麺を食べる必要があるということです。問題文にも書かれていますが、1日でも計画通りの長さの麺を食べることができなければ計画失敗です。
これを判定するために初期値true(計画成功状態)のbool型変数ansと長さXの麺の本数をカウントする写像noodle_countを用意します。そして、\(M\)日分 コードは以下のようになります。 
                
#include 
using namespace std;

int main() {
    vector a(10);
    for(int i = 0; i < 10; ++i) {
        cin >> a[i];
    }
    int x, y, z;
    x = a[0];
    y = a[x];
    z = a[y];
    cout << z << endl;
    return 0;
}
C++で配列を扱うときはvectorが便利です。
また、x = a[0]
を x = a[0] に代入すると
  y = a[a[0]]
これを z = a[y] に代入すると
  z = a[a[a[0]]]
これを用いると以下のように簡略化できます。 
                    
    #include 
    using namespace std;
    
    int main() {
        vector a(10);
        for(int i = 0; i < 10; ++i) {
            cin >> a[i];
        }
        cout << a[a[a[0]]] << endl;
        return 0;
    }
