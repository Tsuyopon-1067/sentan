\(a_0 = 9, a_1 = 8, a_2 = 7, \cdots , a_8 = 1, a_9 = 0\) であると仮定して考えてみます。
初めは画面上に0が表示されているので\(a_9\) に注目します。 1回目の操作で画面に表示される数字は\(a_0\) より9です。 同様に2回目の操作では\(a_9\)より1が、3回目の操作では\(a_{1}\)より8が表示され、答えは8であることがわかります。
実際に実装してみます。\(a_i\)番目を参照する、つまり実行するまでどの数値を見る必要があるのかがわからないときには数列を配列で管理します。
このとき1, 2, 3番目に表示される数字をx, y, zとすると、
x = a[0]
y = a[x]
z = a[y]
となり、答えはzを出力すれば良いです。
コードは以下のようになります。 
                
#include 
using namespace std;

int main() {
    vector a(10);
    for(int i = 0; i < 10; ++i) {
        cin >> a[i];
    }
    int x, y, z;
    x = a[0];
    y = a[x];
    z = a[y];
    cout << z << endl;
    return 0;
}
C++で配列を扱うときはvectorが便利です。
また、x = a[0]
を x = a[0] に代入すると
  y = a[a[0]]
これを z = a[y] に代入すると
  z = a[a[a[0]]]
これを用いると以下のように簡略化できます。 
                    
    #include 
    using namespace std;
    
    int main() {
        vector a(10);
        for(int i = 0; i < 10; ++i) {
            cin >> a[i];
        }
        cout << a[a[a[0]]] << endl;
        return 0;
    }
